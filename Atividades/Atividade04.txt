4 - Crie um dicionário e verifique se uma chave específica existe dentro desse dicionário.

    R:
    dicionario = {'nome':'Luiza Amorim', 'idade':52, 'cidade':'São Paulo'}
    chave = 'idae'
    checador = False

    for dicio in dicionario.keys():
        if chave == dicio:
            checador = True

    if checador:
        print(f'A chave "{chave}" existe dentro desse dicionário')
    else:
        print(f'A chave "{chave}" não existe dentro desse dicionário')


5 - Escreva um código que conte a frequência de cada palavra em uma frase utilizando um dicionário.

    R:
    frase = 'O rato roeu a roupa do rei de Roma e a rainha riu da roupa do rato'
    palavras = frase.split()
    contadorPalavras = {}

    for palavra in palavras:
        if palavra in contadorPalavras:
            contadorPalavras[palavra] += 1
        else:
            contadorPalavras[palavra] = 1

    print(contadorPalavras)
